Function solves a problem of repeating the same items in a list. 

The repetition is understood as more numbers repeating in certain range. As an output, functions returns a list where those repetitive items are excluded. Take a look at \char`\"{}\+Tryout\char`\"{} at the end of this module to understand functionality.
